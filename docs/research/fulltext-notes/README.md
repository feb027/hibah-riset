# Full-Text Reading Notes

These notes are required before Phase 3 final prose uses detailed claims from a source.

## Rule

A source may be cited for high-level background from the ledger, but any specific claim about method, metric, dataset, limitation, or performance must come from a full-text note.

## Required fields per note

- Source ID and bibliographic identity
- Access path / URL / DOI
- Venue and peer-review/index status
- Problem addressed
- Method summary
- Dataset / evaluation protocol
- Metrics reported
- Findings safe to cite
- Limitations stated by authors
- Limitations inferred for this project
- Exact claims allowed in draft
- Claims NOT allowed
- Relevance to sections in `pekerjaan-terkait-outline.md`
